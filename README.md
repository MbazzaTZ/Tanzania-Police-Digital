# Tanzania Police Digital Operations Platform (TPDOP)

A national digital platform for the Tanzania Police Force.

## Stack
- Web Command Center (HTML/CSS/JS — Phase 1)
- Mobile App (React Native — Phase 2)
- Backend: Supabase (PostgreSQL + RLS)
- Auth: Firebase Phone Auth (+255 OTP)

## Modules
- National Dashboard & Command Center
- Citations / Traffic Enforcement
- Arrests & Detentions
- Case Management (CID)
- Intelligence Center
- Wanted & Missing Persons
- Officer & Station Management
- Audit Trail & Reporting

## Role Hierarchy
IGP → DIGP → CP → RPC → SP → OCD → OCS → Officer
